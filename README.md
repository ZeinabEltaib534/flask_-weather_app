# Weather Prediction App

A machine learning web app that predicts weather conditions for Egyptian cities,
built during **NASA Space Apps Cairo Hackathon**.
## Tech Stack
- **Backend**: Flask
- **ML Model**: LightGBM
- **Data Processing**: Pandas, NumPy, Scikit-learn
- **Deployment**: Vercel (Gunicorn)
- **Language**: Python 3

##  Project Structure
flask-weather-app/
├── templates/
├── app.py
├── weather_model.py
├── weather_model.pkl
├── egypt_cities_weather_full_3.csv
└── requirements.txt

##  Installation
```bash
git clone https://github.com/ZeinabEltaib534/flask_-weather_app.git
cd flask_-weather_app
pip install -r requirements.txt
python app.py
```
